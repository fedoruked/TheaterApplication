# TheaterApplication

---

## Requirements
1. PostgreSql
2. .Net Core 3.1
3. Visual Studio
4. Angular 9

---

## How to run
1. Setup ConnectionString in appsettings.json
2. Set startup project TheaterApplication.WebApi and run in Visual Studio
3. Go to folder TheaterApplication.HtmlView
4. Run 'npm i'
5. Run 'ng serve --open'
